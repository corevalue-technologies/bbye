# bbye
Command to kill process, delete user ;-)

# Setup
`$ chmod +x bbye`

you may add these in your path variable via your **~/.bash_profile** file

## Usage of bbye

`$ ./bbye <process/user name>`

e.g.

`$ ./bbye TextEdit`

Okay, Bro. killing...

16551

16579

16584


`$ ./bbye guestuser`

Okay, Bro. deleting guestuser...
