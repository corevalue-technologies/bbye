# bbye
Command to kill process, delete user ;-)

# Setup
`$ chmod +x bbye`

you may add these in your path variable via your **~/.bash_profile** file

## Usage of website-downloader

`$ ./bbye <process/user/filen name>`

e.g.

`$ ./bbye TextEdit`
Okay, Bro. killing...
16551

16579

16584


`$ ./bbye guestuser`
Okay, Bro. killOkay, Bro. deleting guestuser...
